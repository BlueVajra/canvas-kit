Canvas-styled date input

## Installation

```sh
yarn add @workday/canvas-kit-react
```

or

```sh
yarn add @workday/canvas-kit-react-date-input
```

## Usage

```tsx
import * as React from 'react';
import DateInput from '@workday/canvas-kit-react-date-input';

<DateInput />;
```
